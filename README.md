---

### 🗂 할 일 목록 페이지 (`/`)

**목록 조회**

- 로고 클릭 → `/` 페이지로 이동 (새로고침 효과)
- 진행 중인 할 일과 완료된 할 일을 구분하여 표시

**할 일 추가**

- 상단 입력창에 텍스트 입력 후 `추가하기` 버튼 클릭 또는 `Enter` 입력
- 새로운 할 일 생성

**할 일 완료/해제**

- 진행 중 항목의 왼쪽 버튼 클릭 → 체크 표시 & 완료 상태로 변경
- 완료 항목의 왼쪽 버튼 클릭 → 체크 해제 & 진행 중 상태로 변경

---

### 📄 할 일 상세 페이지 (`/items/{itemId}`)

**할 일 수정**

- 항목 클릭 시 상세 페이지로 이동
- 항목 이름 수정 가능
- 할 일 상태(진행/완료) 변경 가능
- 메모 추가 가능
- 이미지 첨부 가능 (최대 1개, 영어 파일명, 5MB 이하)
- `수정 완료` 버튼 클릭 시 반영 후 목록 페이지(`/`)로 이동

**할 일 삭제**

- `삭제하기` 버튼 클릭 시 해당 할 일 삭제
- 삭제 후 목록 페이지(`/`)로 이동

## 🛠 프로젝트 실행 방법

이 프로젝트는 [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app)으로 생성되었습니다.

### 1. 패키지 설치

```bash
npm install
```

### 2. 실행

```bash
npm run dev
```

### 3. 브라우저에서 확인

http://localhost:3000 접속

## 📂 디렉토리 구조 (예시)

```bash

 ├─ page.tsx           # 메인 페이지 (할 일 목록)

 └─ ...
```
